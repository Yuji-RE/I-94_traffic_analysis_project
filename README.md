# I-94州間高速道路における「交通量ピーク」要因の分析

## I-94高速道路の交通量データ分析

- **対象データ**：I-94高速道路の交通量データ
- **目的**：交通量が多くなる条件を特定する
- **主なスキル**：
    - 探索的データ可視化(EDA);
        - 折れ線グラフ・棒グラフ・散布図・ヒストグラム
        - ACFによる自己相関分析・t検定
        - ウェブリサーチ


## プロジェクト概要

本プロジェクトでは、John Hogue氏より提供されている[Interstate 94 westbound; 州間高速道路94号線西行き](https://archive.ics.uci.edu/dataset/492/metro+interstate+traffic+volume)のパブリックデータセットを用いて、デジタルビルボードの広告効果を高めるために、

- 「どのような時間帯・曜日・季節に交通量が多くなるのか」
- 「どのような気象条件のときに交通量が増えたり減ったりするのか」

を探索的データ分析（EDA）によって調べた。

#### 今回の設定（※ 架空の設定）

###### 【役割】
私はとある企業のマーケティング部門に所属する、データアナリストである。

###### 【背景】
来シーズン、弊社より新ゲーム『（データクエスト・ロワイヤル）』が発売される。私は、このゲームの認知度を最大化するため、
データ分析を通じて、I-94高速道路沿い（西向き車線）に新しく設置される予定の複数のデジタルビルボード広告の出稿戦略の一部を担当する。

###### 【課題】
広告代理店から、以下のようなビルボードの放映時間に応じた柔軟な料金プランが提示されている。

- プランA: 24時間放映し続ける（料金は高いが、リーチは最大）
- プランB: 特定の曜日や時間帯に絞って放映する（料金は安いが、リーチは限定的）

⇒ 限られた広告予算の中で最大の効果を得るため、データ分析に基づいて「どの曜日、どの時間帯に広告を集中させるべきか」を
上司に提案する必要がある。

###### 【ミッション】
I-94の交通量データを用いて、**「高速道路の利用が最も多くなるのはどんなときか＝広告効果のゴールデンタイム」**を特定する。


## 結論と課題

###### 【結論】
- 寒い季節に比べて、**暖かい季節（4月～10月ごろ）交通量が多い傾向がある**
- 週末に比べて、平日の方が交通量が多い傾向がある
- 平日の中でも**水・木・金曜日の交通量が多く**、**朝7時と夕方16時ごろがラッシュアワー**である（最大ボリュームは木曜日16時）
- **曇りや軽い悪天候（小雨など）のときは交通量が多い傾向にある**一方で、靄（もや）や霧といった**視界不良となる天候の日は平均交通量が有意に少ない**（約6%差）


###### 【課題】
　今回の分析では「当該ゲームの認知度向上」が主目的なので、「交通量が最も多くなる条件と時間帯」を明らかにした。
しかし、実際に「認知→ 売上への寄与」を果たす為には、自社ゲームのターゲット層についても、具体的なペルソナを分析し設定した上で、
「売り上げを最大化するために、**ターゲット層に最もリーチできる時間帯**」を特定する必要もあった。

そのためには、「I-94高速道路の利用者層」を明らかにした上で、その中でもターゲット層と重なる層が最も広告を目にする
可能性が高い具体的な時間帯と条件の特定、さらに、「**どのような条件で広告を見た時に、購買意欲は高まるか**」という、
別軸での消費者心理の分析も必要になってくる。「広告効果と購買意欲の因果」に対する専門的な洞察を得るため、
先行研究やsnsなどのリアルタイム情報を基に、定量・定性データを行動経済学や社会心理学の観点から分析する事も検討すべきである。

　また、知識不足が故の時系列データの扱いの粗さも目立った。例えば、日中だけのデータに絞って分析をする際、
夜間データを除外したことによる影響を十分に理解しないまま、それを連続データとして時系列プロットにしてしまった。
また、平均の平均をさらに平滑化してしまう処理もあったりして、平均のネスト化により実際よりも変動が小さく抑えられたものを
プロットしていた可能性もある。これらの問題は、今回のような傾向把握を目的としたEDAでは許容範囲かもしれないが、
より厳密なモデル作成や分散評価の際に注意すべき点である。
